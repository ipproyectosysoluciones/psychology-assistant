## Development

### Get Started Developing

```bash
# Clone & install
git clone ...
pnpm install

# Start dev server
npm run dev

# Run tests
npm test

# Lint code
pnpm lint
```

See [DEVELOPMENT.md](./DEVELOPMENT.md) for detailed setup.

---

## Deployment

Deploy to production in minutes:

```bash
# Push to GitHub
git push origin main

# GitHub Actions auto-triggers
# → Build, test, lint
# → Build Docker images
# → Push to registry
# → Deploy (manual trigger available)
```

See [DEPLOYMENT.md](./DEPLOYMENT.md) for step-by-step guide.
